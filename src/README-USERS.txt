CAPE - Child Protection App
===========================

HOW TO START:
1. Double-click: START-CAPE.bat
2. Browser opens automatically
3. Start using CAPE!

REQUIREMENTS:
- Windows 10/11
- Python (usually pre-installed) OR web browser

FEATURES:
- Multi-language (Arabic, French, English)
- Case management
- Dashboard with statistics
- Calendar
- Export/Import data
- 100% Local & Secure
