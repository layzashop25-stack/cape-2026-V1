# Installation Guide

## For Developers

### First Time Setup:
```
1. Run: setup.bat
2. Wait for installation
3. Run: CAPE.vbs
```

### Requirements:
- Node.js 18+
- Any web browser

## For End Users (Distribution)

### Create Package:
```
1. Run: create-package.bat
2. Find: CAPE-Package folder
3. Zip and share
```

### Users Need:
- Windows 10/11
- Python (pre-installed) OR web browser
- No Node.js needed!

## Troubleshooting

**VBS doesn't work?**
→ Run setup.bat first

**Need fresh start?**
→ Delete node_modules folder
→ Run setup.bat again
